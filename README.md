# Interactive Glassmorphism Flipping Card 💳

ဒီ Project ဟာ **HTML5** နဲ့ **CSS3** ရဲ့ 3D Transform feature တွေကို အသုံးပြုပြီး ဖန်တီးထားတဲ့ ခေတ်မီ Glassmorphism ဒီဇိုင်းသုံး Credit Card UI တစ်ခုဖြစ်ပါတယ်။

---

## 🌟 Features (ထူးခြားချက်များ)

* **3D Flip Animation:** Mouse hover လုပ်လိုက်တဲ့အခါ ကတ်က အရှေ့မျက်နှာပြင်ကနေ အနောက်ဘက်ကို 180 ဒီဂရီ ချောမွေ့စွာ လည်သွားမှာဖြစ်ပါတယ်။
* **Glassmorphism UI:** `backdrop-filter: blur()` ကို အသုံးပြုပြီး ဖောက်ထွင်းမြင်ရတဲ့ မှန်သားပြင်ပုံစံ (Frosted glass effect) ကို ဖန်တီးထားပါတယ်။
* **Interactive Design:** Mouse ကို ဖယ်လိုက်တဲ့အခါ မူလအရှေ့ဘက်မျက်နှာပြင်သို့ ပြန်လည်ရောက်ရှိသွားမှာ ဖြစ်ပါတယ်။
* **Realistic Layout:** Mastercard logo၊ EMV chip၊ 16-digit card number နဲ့ အနောက်ဘက်က Magnetic stripe၊ CVV တို့ကို အသေးစိတ် ထည့်သွင်းထားပါတယ်။

## 🛠️ Built With (အသုံးပြုထားသော နည်းပညာများ)

* **HTML5** - Card ၏ ရှေ့မျက်နှာပြင်နှင့် နောက်မျက်နှာပြင် တည်ဆောက်ပုံအတွက်။
* **CSS3** - Glassmorphism effect၊ Flexbox layout နှင့် 3D Rotation animations များအတွက်။

## 🚀 How to Run (အသုံးပြုပုံ)

၁။ ဒီ Repository ကို **Clone** လုပ်ပါ သို့မဟုတ် **Download ZIP** ဆွဲပါ။
၂။ Folder ထဲရှိ `index.html` ကို Browser ဖြင့် ဖွင့်လှစ်၍ Interactive Flip Effect ကို စမ်းသပ်နိုင်ပါသည်။
